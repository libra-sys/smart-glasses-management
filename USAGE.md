# Smart Glasses Management 微信小程序

## 功能介绍

### 1. 设备管理功能

#### 蓝牙搜索与连接
- 自动搜索附近的 `openAIglass` 设备
- 显示设备名称、信号强度等信息
- 支持一键连接设备

#### 设备绑定
- 可以将常用设备绑定到小程序中
- 下次使用时无需重新搜索
- 支持解除绑定设备

#### 电量显示
- 实时显示已连接设备的电量
- 电量变化自动更新

### 2. AI大模型功能

#### 多模型支持
- **OpenAI**：GPT-3.5/4系列模型
- **Anthropic**：Claude系列模型
- **Google Gemini**：Gemini系列模型

#### 自定义API Key
- 用户需要自己购买和配置API Key
- 配置后仅保存在本地
- 支持切换不同模型

#### 组件扩展
- **翻译功能**：快速翻译文本
- **总结功能**：自动总结长文本
- **代码生成**：生成示例代码
- 更多组件可扩展

## 使用指南

### 设备管理

1. **开启蓝牙**
   - 确保手机蓝牙已开启
   - 进入"设备管理"页面
   - 系统会自动检测蓝牙状态

2. **搜索设备**
   - 点击"搜索设备"按钮
   - 等待搜索结果显示
   - 找到需要连接的设备

3. **连接设备**
   - 点击搜索结果中的设备
   - 系统会自动连接
   - 连接成功后显示电量信息

4. **绑定设备**
   - 连接成功后点击"绑定设备"
   - 设备将被保存到"已绑定设备"

### AI大模型

1. **配置API Key**
   - 进入"AI大模型"页面
   - 选择需要使用的模型
   - 输入自己购买的API Key
   - 点击"保存配置"

2. **使用组件扩展**
   - 选择需要的组件（翻译、总结、代码生成）
   - 输入需要处理的文本
   - 点击组件图标自动生成提示词

3. **发送对话**
   - 在输入框中输入问题
   - 点击"发送"按钮
   - 等待AI回复

3. 进入API Keys页面

5. 复制并保存API Key

## 注意事项

1. **蓝牙连接**
   - 确保设备与手机在有效距离内
   - 设备需要处于可配对状态

2. **API Key安全**
   - 请勿将API Key泄露给他人
   - API Key仅保存在本地
   - 建议定期更换API Key

3. **网络连接**
   - 使用AI大模型需要稳定的网络
   - 建议使用Wi-Fi连接

4. **费用说明**
   - 使用大模型会产生费用
   - 请关注各平台的计费规则
   - 建议设置费用提醒

## 扩展开发

### 组件扩展

在 `pages/model/model.js` 中的 `components` 数组添加新组件：

```javascript
{
  id: 'new-component',
  name: '新组件名称',
  enabled: true,
  icon: '🌟'
}
```

### 模型扩展

在 `pages/model/model.js` 中的 `apiConfigs` 数组添加新模型：

```javascript
{
  id: 'new-model',
  name: '新模型名称',
  apiKey: '',
  baseUrl: 'https://api.new-model.com/v1/...',
  enabled: false
}
```

## 常见问题

### Q: 搜索不到设备怎么办？
A: 确保：
1. 手机蓝牙已开启
2. 设备已开机且处于可配对状态
3. 设备与手机在10米范围内
4. 设备名称包含 `openAIglass`

### Q: 连接失败怎么办？
A: 尝试：
1. 重新搜索设备
2. 重启手机蓝牙
3. 重启设备
4. 清除手机蓝牙缓存

### Q: AI没有回复怎么办？
A: 检查：
1. API Key是否正确配置
2. 网络连接是否正常
3. API Key是否有足够的余额
4. 模型是否支持当前请求
